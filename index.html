Removing That Little Gap Under an Image in HTML/CSS

You don't always need to do this. But if you do, here's some good news: it's super easy.

This example page was created by @lswift01.

The Magic Code

Simply add the 'display: inherit' property to the images that you want to remove the gap from:


      img.without-gap {
        display: inherit;
      }
